# Hamza Ben Marouen — Professional Portfolio

An interactive, responsive 3D developer portfolio showcasing the work, skills, and professional experience of Hamza Ben Marouen, a Full Stack & AI Developer.

## 🚀 Live Demo
Visit the live portfolio at: [https://hamzabenmarwen.github.io/Portfolio_hamza/](https://hamzabenmarwen.github.io/Portfolio_hamza/)

---

## 🛠️ Tech Stack

- **Frontend Core**: React 19, Vite, Tailwind CSS, Framer Motion
- **3D Graphics & Effects**: Three.js, React Three Fiber (R3F), Drei, GSAP (GreenSock)
- **Audio & Animations**: Custom audio engine (Passionfruit, Feeling Good), Custom page transition engine (AnimatePresence curtains), Custom cursor
- **Deployment & Delivery**: GitHub Pages (`gh-pages`), EmailJS (for automated contact form delivery)

---

## 🌟 Key Features

1. **Premium 3D Experience**: Smooth, immersive navigation and animated background elements.
2. **Dynamic Transitions**: Custom page transition curtains featuring the page name for fluid browsing.
3. **Interactive Audio**: Immersive ambient background music player.
4. **Comprehensive Showcases**:
   - Detailed pages for each project highlighting tech stack, role, year, and description.
   - Interactive experience timeline including the Master PFE (Assiette Gourmande Sfaxienne), Licence PFE (IT GATE), and Improvement internships.
   - Skill proficiency bars and marquee.
5. **Direct Integration**: Functional, animated contact form powered by EmailJS.

## 💻 Local Development

Follow these steps to run the portfolio on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hamzabenmarwen/Portfolio_hamza.git
   cd Portfolio_hamza
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

---

## 🚀 Deployment

The project is configured for automated deployment to GitHub Pages:

```bash
npm run deploy
```
This command compiles the production bundle using Vite and pushes the output directly to the `gh-pages` branch.
